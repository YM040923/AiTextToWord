# 路线图

这个项目的目标是做一个简单、稳定、适合普通用户使用的 AI 文本整理与导出工具。

## 已完成

- WinUI 3 / Windows App SDK 桌面应用。
- 粘贴 AI 文本并解析 Markdown 结构。
- 导出 Word 和 PDF。
- 标题、段落、加粗、斜体、行内代码、代码块、引用、分割线、列表。
- 字体、字号、行距、页边距、引用样式、列表密度设置。
- 自动读取 Windows 系统字体。
- 独立设置页面。
- 导出后打开文件或打开所在文件夹。
- GitHub Actions 基础构建和测试。
- 一键安装脚本。

## 近期计划

- 补充真实界面截图和导出效果截图。
- 优化 README 中的使用示例。
- 增加更多 Markdown 边界用例测试。
- 改善 PDF 导出排版细节。
- 优化首次安装和升级说明。

## 中期计划

- 支持表格。
- 支持任务列表。
- 支持代码块语言标记和基础高亮。
- 支持更多文档模板，例如学习笔记、正式报告、会议纪要。
- 支持更多导出文件命名策略。
- GitHub Actions 自动生成 MSIX 构建产物。

## 长期计划

- 使用正式代码签名证书，减少测试证书安装步骤。
- 评估 Microsoft Store 发布。
- 更接近 Word 页面效果的应用内预览。
- 多语言界面。

## 暂不计划

- 内置云同步。
- 默认上传用户文本。
- 把 AI 功能做成强依赖。应用应保持“离线也可用”。
